Prérequis :
0 - Télécharger GnuGo : https://gnugo.baduk.org/ (GNU Go 3.8)

1 - Placer le dossier "gnugo-3.8" au même niveau que le backend et le src.

2 - Ouvrir deux terminaux (Windows + R).

3 - Se déplacer sur le projet, et dans l'autre terminal, se déplacer sur le backend.
Exemple :
1er Terminal : D:\Bureau\Jeu-de-GoTest>
2ème Terminal : D:\Bureau\Jeu-de-GoTest\backend>

4 - Taper dans les deux terminaux : npm install

5 - Dans le 2ème Terminal, taper : npm install axios

Pour lancer le jeu :
6 - Dans le 2ème Terminal, taper : node serveur.js

7 - Dans le 1er Terminal, taper : npm run dev

8 - Prendre le lien reçu dans le 1er Terminal et le mettre dans le navigateur.
Exemple :
VITE v5.4.11 prêt en 189 ms

➜ Local : http://localhost:5173/
➜ Réseau : utilisez --host pour exposer
➜ Appuyez sur h + entrée pour afficher l'aide

Vous devez prendre : http://localhost:5173/

Vous pouvez maintenant jouer !

9 - Pour accéder à la page Test, il suffit de faire /Test :
Exemple : http://localhost:5173/Test
La page n'est accessible que de cette manière !
